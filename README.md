## Workflow Git 
Setiap anggota mengikuti alur: 
1. Clone repository 
2. Buat branch `Nama-[NIM]` 
3. Commit minimal 3x 
4. Push branch ke GitHub 
5. Buat Pull Request 
6. Ketua review & merge 
7. Semua anggota `git pull`
